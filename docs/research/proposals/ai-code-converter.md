# AI Code Converter — Research Proposal

**Researcher:** Hettiarachchi S R

## Research Objective

Develop a system capable of translating classical Python code into quantum circuits targeting Qiskit and Cirq frameworks, using transformer-based code generation models.

## Key Research Questions

- How can transformer models be applied to classical-to-quantum code translation?
- What patterns in classical code are amenable to quantum speedup?
- How can quantum suitability be scored and communicated to users?

## Approach

Leverage pre-trained transformer models for code generation, combined with pattern recognition for identifying quantum-amenable algorithms (Grover's, QAOA, Shor's). Implement circuit execution and measurement to validate generated quantum code.
