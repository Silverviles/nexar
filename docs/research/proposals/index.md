# Research Proposals

Each component of Nexar was developed as an independent research contribution. The proposals below outline the research objectives, methodology, and expected outcomes for each component.

| Component | Researcher | Scope |
|-----------|------------|-------|
| [AI Code Converter](ai-code-converter.md) | Hettiarachchi S R | Classical-to-quantum code translation using transformer models |
| [Code Analysis Engine](code-analysis-engine.md) | Jayasinghe Y L | Multi-dimensional code analysis with ML-based classification |
| [Decision Engine](decision-engine.md) | Prasad H G A T | ML-based quantum vs. classical execution routing |
| [Hardware Abstraction Layer](hardware-abstraction-layer.md) | Siriwardhana A H L T S | Provider-agnostic quantum hardware interface |
